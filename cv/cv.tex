\newcommand{\keyprojects}{
  \begin{flushleft}
    \CVSection{Key Projects}
    \project{SecureGateway}{
      is a VPN alternative that eliminates the VPN requirement for our Cloud services to make TCP/UDP
      calls to our customers' sealed network. It is a server-client architecture, which could be running
      in a multi-tenant or single-tenant SaaS environment. SecureGateway is a low-level (L4) network
      shaper \& router, built in fully asynchronous Rust, using gRPC tunnels. The single-tenant version
      is extended by eBPF kernel modules, which can reshape network packets before the Userspace.
    }
    \Sep\\
    \project{WebClients}{
      are Cloud-based SSH \& RDP clients. Partially running in our Cloud infrastructure, and the rendering
      part is running in the User's web browser. SSH terminal buffer updates are rendered by JavaScript code,
      and RDP Graphical PDUs are rendered by a WebAssembly module.
    }
  \end{flushleft}
}
